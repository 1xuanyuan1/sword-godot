# 《仙剑奇侠传》DOS 版完整流程攻略

> 全文剧透。本攻略面向本项目当前识别的 DOS 繁体中文版资源，同时使用简体中文叙述。

## 攻略范围与依据

这是一份从新游戏到结局的主线攻略，包含关键道具、强制谜题、主要 Boss、容易遗漏的支线和当前 Godot 复刻进度。它不是原版对白转储，也不包含原版地图、截图、音乐或其他资源。

攻略主要依据：

- 本机合法导入的 `SCENE`、`EVENTOBJECT`、脚本表、物品表和消息编号；
- 固定 SDLPal 上游 `script.c`、`play.c`、`uigame.c` 等模块的执行语义；
- 项目自动回归实际走过的剧情入口和状态变化。

不同发行版、重制版和民间修改版可能在物品数量、敌人数值、商店价格或个别传送点上不同。本攻略不适用于新仙剑、Steam 重制版或其他改版。

## Godot 复刻版可玩范围

| 标记 | 含义 |

当前 Godot 版已按真实脚本节点自动验证到“交付 36 只傀儡虫并组成灵儿／逍遥／阿奴三人队”。第十二章覆盖剑圣争执、酒剑仙授艺、锁妖塔三场 Boss 与七根盘龙柱、塔毁 RNG、灵岛／月如往事和剑圣黯离；第十三至十五章覆盖神木林、金翅凤凰、大理族议、火麒麟、回魂仙梦、十年前南诏与余杭、水灵珠交换；第十六章继续覆盖两味圣药、灵儿生产、盖罗娇、试炼窟全层、女娲遗迹、傀儡虫真实掉落和土灵珠脱离。生产房、试炼窟外、洞窟各层和女娲遗迹新增 15 个视口后，126 个正式 TileMap 固定画面在 Godot 4.7 Metal 真实窗口下与 CPU 基准达到 320×200 零像素差。100 槽 Godot 存档可保存队伍和完整剧情运行时状态；当前稳定终点为场景 173“李逍遥醒来”、灵儿／逍遥／阿奴 `[1,0,4]` 队伍和入口 `32941`，下一段从五灵珠祭坛开始。TD-003 的经典菜单、物品、六槽装备、剧情商店、场内外仙术、毒和九种状态已经完成，风神召唤、梦蛇变身、普通/召唤合击与特殊物品脚本也进入真实结算路径；ScriptVM 的真实可达操作码审计已全部通过。后续主线仍需逐段验证大理之战、南诏终战和结局音画，因此暂不能仅据系统覆盖宣称 Godot 版已经可以通关。

## 通关前的通用建议

1. 进入迷宫前另存一档，离开城镇前补足回复药、解毒药和驱邪类道具。
2. 新队员入队后检查装备。原版很多装备不会自动换上，物品在背包里不等于已经生效。
3. 李逍遥学会“飞龙探云手”后，可在不影响胜利的前提下先偷 Boss 或稀有敌人；完整目标与奖励见[飞龙探云手偷窃图鉴](FLYING_DRAGON_STEAL_GUIDE.md)，本攻略不把偷窃奖励列为通关必需品。
4. 五灵珠既是重要道具，也是终盘祭坛的必需品。不要因为战斗效果强就误以为可以丢弃；原版会一直保留到祭坛脚本消耗。
5. 迷宫遇到岔路时先清一侧再返回。将军冢、锁妖塔、神木林和试炼窟尤其容易因为连续换层失去方向感。
6. 剧情战前保持全员较高体力，优先处理毒、封、眠、乱等异常。无法判断属性时，先用普通攻击和无属性技能观察。

## 第一章：余杭客栈、桂花酒与买虾 ✅

1. 开场梦境结束后，按剧情与李大娘对话。
2. 下楼招待三名黑苗客人，再去厨房查看酒菜。
3. 把酒菜送到客房。黑苗客人嫌酒不好，桂花酒会回到李逍遥手中。
4. 离开客栈，在村外找到醉道士。面对他使用桂花酒，完成喝酒剧情。
5. 醉道士约李逍遥夜晚到十里坡山神庙见面。
6. 返回客栈。李大娘给 50 文，让李逍遥买鲜虾。
7. 找鱼嫂询问，得知风浪影响，没有鲜虾可卖。
8. 返回客栈，发现李大娘病倒，主线转为寻找仙药。

初始客栈可搜索的物品：

- EventObject 6：皮帽；
- EventObject 7：木鞋。

它们是地图物件上的搜索点，不是独立人物 Sprite。面对对应柜子或物件按搜索键，拾取后事件会隐藏，物品进入背包。

## 第二章：仙灵岛与水月宫 ✅

1. 在余杭打听治病方法，确认仙灵岛可能有仙药。
2. 与黑苗人交谈，取得破天锤并服下忘忧散。黑苗人要求李逍遥破坏岛上的石像。
3. 找船夫前往仙灵岛。出发前整理物品，岛内流程较长。
4. 登岛后依次寻找六尊石像，面对石像使用破天锤。前五尊会打开通路，第六尊破碎后破天锤也会损坏。
5. 沿打开的水路和荷叶通道前往水月宫区域。
6. 触发赵灵儿沐浴剧情，按约定归还衣物，随后进入水月宫。
7. 姥姥发现外人进入，剧情会推动李逍遥与赵灵儿成亲。
8. 次日取得救治李大娘所需的紫金丹，乘船返回余杭。
9. 忘忧散开始生效，李逍遥对岛上经历产生遗忘。

关键道具：

- 破天锤：石像机关专用；
- 紫金丹：救治李大娘的主线物品，物品编号 `273`；
- 忘忧散：剧情道具，无需玩家主动解毒。

## 第三章：山神庙、仙灵岛惨案与离开余杭 ✅

1. 把紫金丹交给李大娘，完成服药剧情。
2. 夜晚从客栈离开，沿十里坡前往山神庙。
3. 与醉道士会合，观看御剑术教学。不要跳过该剧情，否则后续技能和故事衔接会异常。
4. 返回客栈休息。次日根据村民和小虎提供的消息，再次前往仙灵岛。
5. 水月宫已遭黑苗人袭击。找到赵灵儿并让她入队。
6. 返回余杭客栈，触发黑苗人冲突和强制战斗。
7. 战斗后与李大娘告别，整理她交给李逍遥的行装。
8. 到码头乘船前往苏州。离港前完成余杭尚未拾取的皮帽、木鞋和其他搜索点。

## 第四章：苏州城与林家堡 ✅

1. 苏州城外遇到林月如责打长工。介入后按剧情与林月如交手。
2. 离开一小段距离后听见呼救，返回救下遭山贼袭击的林月如，再完成后续误会战斗。
3. 进入苏州城，补给并收集城内消息。算命摊、客栈和比武招亲传闻会为后续剧情提供提示。
4. 前往林家堡，参加比武招亲并击败林月如。
5. 入夜后赵灵儿因身体变化离开。追到林家堡后院，剧情误会令林月如加入队伍。
6. 离开苏州，沿城外道路寻找赵灵儿的去向。

战斗建议：

- 前几场林月如战以保命为主，及时回复即可；
- 比武招亲前先更新武器、防具和鞋子；
- 赵灵儿暂时离队前，不要把唯一的重要回复物全部留在她身上。

## 第五章：隐龙窟 ✅

1. 在苏州城外打听失踪少女，确认她们被带往隐龙窟。
2. 进入洞窟后沿主通道向深处走，优先开启沿途宝箱并记住上下层入口。
3. 击败蛇妖男后继续深入。
4. 在洞窟深处击败狐妖女，释放被抓的少女。
5. 离开隐龙窟，继续前往白河村。

建议准备解毒与群体恢复。洞内敌人密集，如果法力不足，先退回入口外整备，不必强行连续推进两个 Boss。

## 第六章：白河村六味药材 ✅

赵灵儿被安置在韩医仙家中。完整药方共有六味，韩医仙能准备其中三味，玩家必须取得以下三样：

1. 银杏子：到韩医仙家后院的银杏树搜索或采摘。
2. 活鲤鱼：在村内民居向渔夫借钓竿，到河边面对钓鱼点使用，取得鲤鱼后归还钓竿。
3. 鹿茸：到白河村前山路拾取捕兽夹，面对鹿的行进路线使用，再把鹿赶入夹子；取得鹿茸后会放鹿离开。

收集完成后返回韩医仙家。脚本会确认银杏子、鹿茸、鲤鱼齐全，配药并推进赵灵儿恢复剧情。

开发对照：药方对白称“银杏子”，实际采集进背包的是银杏果；银杏果、鲤鱼、鹿茸分别对应物品 `281`、`282`、`283`。脚本 `14700–14702` 会连续消耗三者；消息表明确要求的另外三味为人参、雪莲子、何首乌。

## 第七章：玉佛寺、黑水镇与将军冢 ✅

1. 从白河村西北方向前往玉佛寺。
2. 在寺内调查和尚异常，推进与智修大师、小和尚智泽有关的剧情。
3. 完成战斗后，小和尚恢复为玉佛珠并加入队伍道具。
4. 前往黑水镇，先开启镇内对象 1215 的宝箱取得引路蜂。到乱葬岗墓地的尸妖封锁前，必须让赵灵儿实际装备玉佛珠；只放在背包里不会生效。
5. 在镇内调查通往地下的入口，进入将军冢。
6. 将军冢多次上下换层。建议遇岔路先固定向一个方向探索，并用宝箱是否已开启判断来路。
7. 击败鬼将军后不要原路离开，继续寻找通往地下血池的道路。
8. 在血池击败赤鬼王，取得土灵珠。
9. 赤鬼王战后会开放血池场景传送。使用引路蜂离开地下区域，再沿墓地、乱葬岗、黑水镇和后山原路返回白河村；土灵珠是后续关键物品，不会在返程时消耗。

关键检查：没有击败赤鬼王并取得土灵珠，后续鬼阴山和终盘五灵珠祭坛都会缺少必要状态。

开发对照：当前 Godot 真实资源回归已完成本章全程。玉佛寺依次使用敌队 28/35、战场 15；鬼将军为敌队 26／战场 18，赤鬼王为敌队 27／战场 19。玉佛珠为物品 `274`，墓地入口 `0086 274,0` 的零数量操作数仍表示至少装备一件；土灵珠为 `267`，引路蜂为 `151`。返程后诊厅消息 `4405–4425` 会打开下一章赵灵儿被掳入口。

## 第八章：鬼阴山与石长老 ✅

1. 返回白河村后得知赵灵儿被带走，方向指向西南方鬼阴山。
2. 进入鬼阴山，穿过山道与据点，沿黑苗守卫最密集的路线前进。
3. 在鬼阴坛推进赵灵儿、石长老和盖罗娇相关剧情。
4. 石长老相关战斗带有强烈剧情性质。保证队伍能撑过攻击和异常，不要把“必须在此击杀某角色”当作解谜目标。
5. 剧情结束后，按出口路线前往扬州。

开发对照：当前 Godot 真实资源回归已完成本章全程。前路守卫为敌队 33／战场 52，石长老剧情战为敌队 34／战场 16；长剧情会卸下并移除玉佛珠 `274`，把队伍暂时缩为李逍遥单人，随后在得救场景恢复李逍遥普通 Sprite 2 并组成李逍遥／林月如 `[0,2]` 双人队。后门经场景 77/79 最终抵达场景 82“山道·扬州前”，土灵珠 `267` 保留。

## 第九章：扬州女飞贼案 ✅

1. 进入扬州后城门因女飞贼案封锁，先在客栈投宿。
2. 夜间发生失窃，立刻沿屋顶追赶女飞贼。屋顶岔路以能继续向上或向右跨越的路线为主，走错可退回上一个屋面。
3. 追回财物后返回客栈，随后林月如会因误会被官府拘押。
4. 到衙门了解案情，再调查城内井口、密道和女飞贼藏赃处。
5. 在地下区域找到证据并击败真正的女飞贼。
6. 返回衙门洗清林月如嫌疑。
7. 城门重新允许通行后离开扬州，继续向京城方向前进。

容易卡住的点：屋顶追逐和井下密道是同一案件的两个阶段；只完成屋顶追逐并不等于案件结束。

开发对照：当前 Godot 真实资源回归已完成本章全程。夜间客栈与天亮屋顶两次使用敌队 29，战场分别为 21/30；真凶为敌队 30／战场 16，对象 `[526,479,526]`。布包 `271` 必须在屋顶取得、面对古董商使用后再由交还剧情消耗；首次从井底取证会进入受审和林月如离队分支，探监脚本实际收取 300 文，复审后悬赏增加 5500 文。北门入口最终抵达场景 104“山路·蛤蟆谷前”，队伍恢复为 `[0,2]` 并保留土灵珠 `267`。

## 第十章：蛤蟆山与金蟾鬼母 ✅

1. 从扬州往京城方向进入蛤蟆山区域。
2. 沿谷地前后段前进，遇见受伤女子后进入她所在的蛤蟆洞；第一次交谈后按剧情先退出洞穴，李逍遥与林月如会察觉疑点。
3. 准备足够解毒药，再次进入洞穴，击败金蟾鬼母和蛤蟆精。
4. 战后调查蛤蟆精尸体，取得五毒珠。该道具会保留到后续流程，并能显著降低毒系战斗压力。
5. 穿过洞穴后段抵达白苗酒店。盖罗娇会以迷药让李逍遥和林月如昏睡，店外随即发生白苗与石长老的冲突。
6. 剑圣带走赵灵儿后，回到店外救醒盖罗娇，得知灵儿暂时被剑圣所救；队伍恢复为林月如／李逍遥双人。
7. 经长安城外前往水仙尊王庙码头，与船家交谈并遇见尚书夫人，随后乘船进入长安。

开发对照：当前 Godot 真实资源回归已完成本章全程。金蟾鬼母与蛤蟆精为敌队 36／战场 18，对象 `[500,465]`；尸体脚本取得五毒珠 `262`。白苗酒店使用剧情敌队 37／战场 20，对象 `[496]`，战斗中临时换为剧情角色，结束后恢复李逍遥普通 Sprite 2 和双人队。盖罗娇获救会把码头船家入口从 `19916` 改为 `19918`，乘船后进入场景 99“长安”，完成首次入城对白并保留五毒珠、土灵珠。

## 第十一章：京城尚书府与彩依 ✅

1. 到刘尚书府拜见云姨，再经后院探望病重的刘晋元；晚宴后回到长安街头继续打听彩依的来历。
2. 向街头乞丐购买彩依的传闻时，脚本会收走当前现金的一半；随后返回尚书府，经历林天南的七诀剑气剧情战。
3. 再次进入刘晋元房间，目睹彩依强迫他服药。刘晋元逃到院外求助后，云姨会给三万文；城中的茅山道士实际收取一万五千文，但第一次作法会失败。
4. 从长安运河救出酒剑仙并请他开坛。作法需要：

   - 蜡烛；
   - 符纸；
   - 檀香；
   - 一壶酒。

5. 檀香、蜡烛和符纸可以从尚书府楼上的柜子取得；酒可在长安酒楼花一百文购买。备齐后交给酒剑仙，开坛剧情会消耗四件物品。
6. 法事中林月如擅自离开，随后在刘晋元居所外发现蝶精彩依。完成剧情战后，从尚书府后门追入毒仙林。
7. 在蜘蛛巢穴击败毒娘子。酒剑仙会从蜘蛛精体内取出雷灵珠，彩依则以千年修为换取刘晋元十年寿命。
8. 三段往事回忆结束后，随酒剑仙御剑抵达蜀山前山。

开发对照：当前 Godot 真实资源回归已完成本章全程。林天南为敌队 38／战场 7，对象 `[525]`；蝶精彩依为敌队 39／战场 28，对象 `[468]`；毒娘子为敌队 42／战场 27，对象 `[435]`。连续状态覆盖三万文援助、一万五千文道士费用、物品 `81/82/83/86` 的取得与消耗、单人／双人队切换、场景 `99/107/114–125/137–142/154`、三段回忆及 RNG #3 御剑过场，最终以李逍遥／林月如 `[0,2]` 队伍抵达场景 154“蜀山·前山”。雷灵珠 `264` 此时由酒剑仙保管，尚未进入背包；它会在下一章决定闯锁妖塔后正式交付。

## 第十二章：蜀山与锁妖塔 ✅
