# PROD 版本更新日志

## 版本记录

## 0.1.1

- **发布日期**: 2026-07-23
- **发布环境**: PROD
- **版本类型**: release
- **版本说明**: 为 Web 与 Android 增加完整触屏操作，移动设备无需外接键盘即可游玩。
- **涉及模块**: 地图探索、对话、经典菜单、战斗、Web 发布、Android APK

### 更新内容

1. 地图自由探索增加按下才出现的浮动摇杆、左上菜单入口和右下互动键，松手或进入模态流程会立即释放移动状态。
2. 标题、存档、经典菜单和对话支持直接点击，并在移动菜单右上角提供统一返回入口。
3. 战斗支持点击四向指令、仙术／物品列表和敌我目标，保留原有键盘输入。
4. Web 触屏环境与 Android 自动启用移动操作；Android 验收包升级到 `0.1.1(2)`。

## 0.1.0

- **发布日期**: 2026-07-23
- **发布环境**: PROD
- **版本类型**: release
- **版本说明**: 发布可从主线开篇持续游玩至完整结局的 Godot Web 版本。
- **涉及模块**: 完整主线、TileMap 地图、经典战斗、音画、Web 发布

### 更新内容

1. 完成从新游戏到拜月教主／水魔兽终战及 DOS 结局的连续主线流程。
2. 正式地图统一使用 TileMapLayer 渲染，覆盖全部有效地图、场景引用、人物动作与遮挡规则。
3. 提供经典菜单、物品／装备／仙术、回合制战斗、剧情动画、音乐音效及版本化存档。
4. Web 包使用外部分片加载完整游戏内容，并修复网页环境的内容目录和状态图标依赖。
